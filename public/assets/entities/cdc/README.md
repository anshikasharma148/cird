# CDC Facility Images

This directory contains images of CDC facilities and rooms.

## Image Naming Convention

Place CDC room images with the following naming convention:

- `cdc.jpg` - Main CDC image (used in header/hero section)
- `reception.jpg` - Reception area
- `control-room.jpg` - Control Room
- `engineering-room.jpg` - Engineering Room
- `shift-engineers-room.jpg` - Shift Engineers Room
- `server-room.jpg` - Server Room
- `dr-dhananjay-office.jpg` - Dr. Dhananjay R. Mishra Office
- `dr-amit-office.jpg` - Dr. Amit Kumar Srivastava Office
- `conference-room.jpg` - Conference Room
- `executive-room.jpg` - Executive Room

## Image Specifications

- **Format**: JPG, JPEG, or PNG
- **Aspect Ratio**: 16:9 (landscape) recommended for slider
- **Size**: Minimum 1920x1080 pixels, recommended for high quality display
- **Optimization**: Compress images for web to ensure fast loading

## Notes

- Images will automatically display in the slider on the CDC page
- If an image is missing, a placeholder will be shown
- Images should be professional and showcase the facilities well
- Supported formats: JPG, JPEG, PNG, WebP
