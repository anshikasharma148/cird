# MTL Lab Images

This directory contains images of MTL Lab facilities.

## Image Naming Convention

Place MTL Lab images with the following naming convention:

- `image-1.jpg` - MTL Lab Image 1
- `image-2.jpg` - MTL Lab Image 2
- `image-3.jpg` - MTL Lab Image 3
- ... (continue numbering as needed)
- `image-10.jpg` - MTL Lab Image 10

You can add more images by continuing the numbering sequence.

## Image Specifications

- **Format**: JPG, JPEG, or PNG
- **Aspect Ratio**: 16:9 (landscape) recommended for slider
- **Size**: Minimum 1920x1080 pixels, recommended for high quality display
- **Optimization**: Compress images for web to ensure fast loading

## Notes

- Images will automatically display in the slider on the MTL page
- If an image is missing, a placeholder will be shown
- You can update room names later in the code when details are available
- Supported formats: JPG, JPEG, PNG, WebP
