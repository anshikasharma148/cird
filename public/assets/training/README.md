# Training Images Directory

This directory contains images for the Training page galleries.

## Folder Structure

Place your images in the following folders with the naming convention shown:

### `srijan/` - AI Srijan FDP Images
- Place 6 images for the AI Srijan FDP gallery
- Naming: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`, `6.jpg`
- Recommended format: JPG or PNG
- Recommended aspect ratio: 16:9 (landscape)

### `jnstpp/` - Jaypee Nigrie Super Thermal Power Plant Images
- Place 8 images for the power plant visit gallery
- Naming: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`, `6.jpg`, `7.jpg`, `8.jpg`
- Recommended format: JPG or PNG
- Recommended aspect ratio: 1:1 (square)

### `closing/` - Closing Ceremony Images
- Place 12 images for the closing ceremony gallery
- Naming: `1.jpg`, `2.jpg`, `3.jpg`, ... `12.jpg`
- Recommended format: JPG or PNG
- Recommended aspect ratio: 1:1 (square)

## Notes
- Images will automatically load when placed in the correct folders
- If an image is missing, a placeholder will be shown
- Images should be optimized for web (compressed) for better performance
- Supported formats: JPG, JPEG, PNG, WebP
